Download Enlightenment


Main Repository

All EFL code is hosted publicly on the code SVN repository and can be freely downloaded and distributed. There are a number of useful examples included in most of the source directories. There are also python, C++, perl, javascript and various other bindings available for ease of writing.


Exchange

A large database of user made themes, applications and modules can be found at Exchange. This is the best place to find latest updates to themes, and also 3rd party applications that are hosted on websites other than the main SVN repository.


Main Website

The main website for Enlightenment holds much documentation on EFL usage, as well as information on contributing to the project, and a bug tracker for the public to submit problems and wishes.

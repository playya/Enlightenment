Enlightenment, create lean and beautiful software

Developers empowered

  • The API is intuitive and spans a broad number of uses while remaining consistent. From interface design to advanced data types and system interaction.
  • The interface development approach uncouples design an development at every level making your applications themeable without extra effort.
  • Written in C99 with an emphasis on portability and performance. Even the high-level language bidings are suitable for both desktop and embedded development.

learn more

Artists unleashed

  • Every aspect of interface design can be covered with a language syntactically similar to CSS but capable of creating arbitrary design elements at the designer's whim.
  • Transitions (and triggers) defined by the designer let him improve usability or create aesthetically pleasing animations. Without touching a line of application code.
  • Applications and utilities to aid designers are either already available or under development, from simple viewers to WYSIWYG editors.

Featured Software

Featuring the E17 desktop shell. I did a new snapshot release of E17 + Core EFL. This includes Eet 1.1.0, Ecore 0.9.9.050, E_DBus 0.5.0.050, Edje 0.9.9.050, Efreet 0.5.0.050, Embryo 0.9.9.050 and Enlightenment 0.16.999.050. Lots of improvements and fixes. Too many to count.

Eet 1.1.0 released

Eet 1.1.0 has been released. This release includes the following changes from 1.0.0: Fixes to data codecs (data descriptors). Fixes for non-existing files opened READ/WRITE and other miscellaneous bugs found. Speedups for decoding and lookups, make inline-image decode better on some platforms and add signed-eet file support as well as opening eet files from memory.

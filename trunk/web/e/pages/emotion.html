Emotion is an Evas smart-object library providing video capabilities. Emotion leverages libxine 1.0 and integrates seemlessly with the rest of the EFL. Because it's based on libxine, any format that it supports (Theora, DiVX, MPEG2, etc) is avalible using Emotion.

To demonstrate just how simple Emotion is to use, please look at this DVD player written in 18 lines of C. 

#include 
#include 
#include 
#include 

Evas_Object *video;

/* the main function of the program */
int main(int argc, char **argv)
{
   Ecore_Evas *ee;

   /* create a canvas, display it, set  */
   ecore_evas_init();
   ee = ecore_evas_software_x11_new(NULL, 0, 0, 0, 800, 600);
   ecore_evas_show(ee);

   /* create a video object */
   video = emotion_object_add(ecore_evas_get(ee));
   emotion_object_file_set(video, "dvd:/");
   emotion_object_play_set(video, 1);
   evas_object_resize(video, 800, 600);
   evas_object_show(video);

   /* run the main loop of the program - playing, drawing, handling events */
   ecore_main_loop_begin();
   
   /* if we exit the main loop we will shut down */
   ecore_evas_shutdown();
}

With Epsilon it's unbelievably easy to add video to your application, and by leveraging the rest of the EFL to build more powerful video players than seen anywhere else.

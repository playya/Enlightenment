Welcome to the Enlightenment Project!

Enlightenment is a Window Manager for the X Window System, traditionally used in Unix environments. On this site you'll find information about both the current release of Enlightenment and what's happening regarding the development version, including the Enlightenment Canvas Library (Evas), the imaging library Imlib2, the Enlightenment File System Daemon (EFSD), and the other components of the Enlightenment desktop shell. If you want to jump right in, you can download everything you need here.

Currently, for support, you have three options: using irc (preferred; #e on irc.openprojects.net), our mailing lists, or email (use email addresses from the team page, but don't bug Raster if you don't need to). Suggestions are welcome.

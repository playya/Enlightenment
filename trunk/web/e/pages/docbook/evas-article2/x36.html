2. Lines

Lines are created with this function:

Evas_Object evas_add_line(Evas canvas);

The 'canvas' parameter indicates the canvas the line shall be drawn on. A line can thus only belong to a single canvas at a time. The value returned is an opaque structure that represents the line object.

Actually, an Evas_Object can point on any type of object that Evas can manipulate. This makes it possible to have only one function to change to color of an object, whatever its type is. That function is:

void evas_set_color(Evas        canvas,
                    Evas_Object object,
                    int         red,
                    int         green,
                    int         blue,
                    int         alpha);

The 'object' parameter is the object which color will be changed. 'canvas' is the canvas that displays the object. The parameters 'red', 'green' and 'blue' define the objet's color components. Their values must be between 0 and 255. The 'alpha' parameter sets the opacity of the object, and its value is also between 0 (completely transparent) and 255 (completely opaque).

After its creation, a line start on point (0,0) and ends on point (0,0), which isn't very interesting. The following function is used to set the coordinates of a line's extremities:

void evas_set_line_xy(Evas        canvas,
                      Evas_Object line,
                      double x1, double y1,
                      double x2, double y2);

After a call to this function, the 'line' object of the 'canvas' canvas will be drawn from point (x1,y1) to point (x2,y2).

However, objects created by Evas and placed on a canvas are by default invisible. Use the following function to make them visible:

void evas_show(Evas canvas, Evas_Object object);

If afterwards an object must be temporarily hidden the corresponding function is:

void evas_hide(Evas canvas, Evas_Object object);

That function only makes the object invisible. It keeps its properties (colors, size, etc) and uses up memory. To remove it completely from memory (it will then not be usable anymore) use this function:

void evas_del_object(Evas canvas, Evas_Object objet);

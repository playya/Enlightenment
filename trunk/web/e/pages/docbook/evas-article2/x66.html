4. Polygons

Evas can also display polygons without restrictions on the number of sides. A polygon is added to a canvas with the function:

Evas_Object evas_add_poly(Evas canvas);

The polygon created will not have any vertex and thus does not really exist. The following function adds a point to a polygon:

void evas_add_point(Evas canvas,
                    Evas_Object object,
                    double x, double y);

The 'x' and 'y' parameters are the coordinates of the point we wish to add to the object 'object' on the canvas 'canvas'.

To change the coordinates of the vertices of a polygon, the only solution is to remove all the points and re-add the vertices with the new coordinates with evas_add_point() function calls. The following function removes all the vertices of a polygon:

void evas_clear_points(Evas canvas,
                       Evas_Object object);

6. Text

Evas can also display text, automatically using antialiasing if it's needed. This function is used to displayed text on a canvas :

Evas_Object evas_add_text(Evas canvas, char *font,
                          int size, char *text);

The 'text' parameter is the text that the new object will display. 'size' is the size in points of the font. The font is chosen by the 'font' parameter: it is the name of the font, without the suffix. For example, to display text with "andover.ttf", the 'font' parameter to use is simply "andover".

Evas searches for the requested font in a list of directories. To add a directory to this list, use this:

void evas_font_add_path(Evas canvas, char *directory);

Note that each directory list is tied to a single canvas.

A directory can also be removed from the list with this function:

void evas_font_del_path(Evas canvas, char *directory);

It is possible to change the text of a text object after its creation with this function:

void evas_set_text(Evas canvas, Evas_Object object, char *text);

It is possible as well to change the size and the font of a text object:

void evas_set_font(Evas canvas, Evas_Object object,
                   char *font, int size);

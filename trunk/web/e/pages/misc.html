Miscellaneous Applications

Enlightenment is comprised of many libraries. These libraries provide a lot of functionality, and are not intended for exclusive use by E17. This page lists other applications that use these libraries.

To have your E-based application added to this page, let us know on the mailing list.

Entice
Author: Raster
Download: http://www.nl.rasterman.com/projects/entice/index.html
Description: An simple image viewer based on Evas and Ecore.
 
E_Snake
Author: Martin Sarajervi
Download: http://ms.klmoberg.no/e_snake.tar.gz
Description: An evas-based implementation the Snake game.

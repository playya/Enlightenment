Welcome. Your quickly on your way to getting started with Enlightenment DR0.16.5. This release has been considered stable since October 27th, 2000. Before you get E, review the following list of required libraries and applications:

    Enlightenment DR0.16.5 Requirements:
  • IMLIB v1.9.8
      IMLIB Requires:
    • GTK+ 1.2.1 (or greater)
    • GLIB 1.2.1 (or greater)
    • libTIFF 3.5.5
    • libPNG 1.0.8
    • Zlib 1.1.3
    • libJPEG 6b
    • libUNGIF 4.1.0 (optional)
  • FNLIB 0.5 (or greater)
  • FreeType 1.3.1 (or greater)

Finally, Enlightenment is probably considered the most visually appealing window manager out there. One way it does this is with "themes", which it includes a couple of with it's regular packages. If you'd like to try different ones out, one of the best sites for finding new ones is e.themes.org.

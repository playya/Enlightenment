Documentation

Enlightenment 0.17

You will find documentation on how to use and configure E 0.17 here when the time's right. At the moment, E 17's codebase is not considered ready for public consumption. IF you try E 17, please do NOT come to the mailing lists with questions like "Nothing happens!", "Why doesn't feature XYZ exist" etc.

Development Documentation
NOTE: Work on the documentation is still in progress and therefore it may be incomplete.

The E Programming Manual: By Christian Kreibich [HTML]
This document explains the Enlightenment 0.17 release, for both users and developers.

The Ebg Programming Manual: By Rasterman and Atmos [HTML]
Ebg is a background library for Enlightenment 0.17. It offers a creative new way approach to backgrounds in your desktop environment.

Evas Programmers Guide: By Rasterman [PDF] [HTML]
This is a comprehensive document written by the lead developer to explain everything you want to know about the Evas canvas library. Includes excellent introduction, good discussion, complete API reference, including five complete programming examples and numerous rendering samples for illustration.

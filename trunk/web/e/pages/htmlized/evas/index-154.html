void evas_callback_del(Evas e, Evas_Object o,
Evas_Callback_Type callback);
This function removes all callbacks set on the specified object that is of the type
callback
, if
there are any.
Arguments:
e
A valid Evas canvas handle.
o
A valid object handle in the canvas specified.
callback
The callback type. This can be one of:
CALLBACK_MOUSE_IN, CALLBACK_MOUSE_OUT, CALLBACK_MOUSE_DOWN,
CALLBACK_MOUSE_UP, CALLBACK_MOUSE_MOVE, CALLBACK_FREE.

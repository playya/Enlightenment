void evas_free(Evas e);
This function will free all objects in the Evas canvas passed to it, free the canvas and all
memory used by that canvas. If Evas created the window for you for this canvas with
evas_new_all(), this window will also be destroyed.
Arguments:
e
A valid Evas canvas handle.

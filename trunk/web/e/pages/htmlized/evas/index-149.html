int evas_pointer_in(Evas e);
This function returns if the mouse cursor is currently considered by the canvas to be inside
it. IT returns 1 if it is, and 0 if it is not.
Return:
It returns 1 if the pointer is considered to be in the canvas, and 0 if it is not.
Arguments:
e
A valid Evas canvas handle.

Code Examples
The following snippets of code are examples of how to use Evas, and to demonstrate how easy it
really is. Note that these examples require the Evas library AND the Ecore library that wraps and
handles X Events for us so we don't have to do as much X coding as we would normally have to to
get these programs to run. You will notice the Ecore routines start with an e_ notation.

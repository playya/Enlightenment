char *evas_get_text_font(Evas e, Evas_Object o);
This function returns the font name used by a text object. If the object is not a text object, an
empty string is returned. The font name returned is only valid so long as the object isn't
freed or the font isn't changed.
Return:
A pointer to a C string (0 byte terminated) that contains the font name of the object.
Arguments:
e
A valid Evas canvas handle.
o
A valid object handle in the canvas specified.

Tinting and Fading
Doing effects in Evas is almost child's play like everything else. You can tint and fade image objects,
for example, with very little effort. Let's start a new Evas with a single image object and a textured
background.
Now lets "tint" this object red. That's easy:
evas_set_color(evas, object, 255, 0, 0, 255);
When you set the colour of an image, every pixel of that image is multiplied by the colour value. So
pseudo-code wise output.red = (pixel.red * color.red) / 255, and the same for the green, blue and

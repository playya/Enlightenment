void evas_unset_clip(Evas e, Evas_Object o);
This routine disables clipping for the object specified. After this the object will no longer be
clipped by any clip object and will be displayed as if it were never clipped in the first place.
Arguments:
e
A valid Evas canvas handle.
o
The object to remove the clipping from.

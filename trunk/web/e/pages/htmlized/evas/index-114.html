Evas_Object evas_object_get_named(Evas e, char *name);
This function returns the object handle of an object whose name matches the specified
name parameter passed in. There must be an object whose name matches the name
specified to get back a valid object handle, otherwise NULL is returned. If there is more than
one object with such a name it is undefined as to which object handle will be returned.
Return:
A valid object handle of the object that has the name requested, or NULL if none exists in
the specified canvas.
Arguments:
e
A valid Evas canvas handle.
name
A C string (0 byte terminated) that contains the name being queried.

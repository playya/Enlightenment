int evas_get_font_cache(Evas e);
This function returns the size of the font cache for the engine used by the specified canvas
in bytes.
Return:
An integer equal or greater than 0, that is the number of bytes allowed to be used up as
cache for non visible fonts.
Arguments:
e
A valid Evas canvas handle.

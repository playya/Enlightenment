void evas_event_button_up(Evas e, int x, int y, int b);
This function passes a mouse button raise event to the Evas canvas specified. As with all
the event functions, callbacks are synchronous and will be called if they are triggered the
moment this function is called, and it will not return until all callbacks triggered have
returned.
Arguments:
e
A valid Evas canvas handle.
x, y
Pixel space co-ordinates where the mouse button was raised.
b
The button number that was raised. Can be a value from 1 to 32 only.

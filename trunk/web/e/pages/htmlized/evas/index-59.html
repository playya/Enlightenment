void evas_set_output_viewport(Evas e, double x, double y,
double w, double h);
This function is used to tell Evas what geometry the output viewport has. By default when
you first create an Evas canvas, the viewport geometry is at 0,0 for its top-left corner and its
width and height in units is the same as the output canvas size in pixels. Thus we have a 1
to 1 canvas co-ordinate space to pixel space ratio by default. You can change its size and
location whenever you want. This will result in effectively being able to zoom into or scroll
around the canvas.
Arguments:
e
A valid Evas canvas handle.
x
The top-left x co-ordinate in canvas units for the box that defines the viewport region.
y
The top-left y co-ordinate in canvas units for the box that defines the viewport region.
w
The width of the viewport in canvas units (must be greater than 0).
h
The height of the viewport in canvas units (must be greater than 0).

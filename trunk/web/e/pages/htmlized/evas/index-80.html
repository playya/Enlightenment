void evas_set_text(Evas e, Evas_Object o, char *text);
This function sets the text string contents of a text object to the text supplied in the
arguments. This has no effect on objects other than text objects.
Arguments:
e
A valid Evas canvas handle.
o
The Evas object handle of a text object to modify the textl of.
text
The C string (0 byte terminated array of chars) containing the ascii text to use for the text
object.

Evas_Object evas_add_rectangle(Evas e);
This function creates a rectangle object in the specified Evas canvas. The default colour is
0,0,0,0 and default location is 0,0 with a default size of 1 x 1 units in size. The program still
needs to show, move, resize, set the colour etc. of the object.
Return:
A valid Evas object handle.
Arguments:
e
A valid Evas canvas handle.

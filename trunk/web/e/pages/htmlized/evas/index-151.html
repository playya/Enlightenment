int evas_pointer_buttons(Evas e);
This function returns a mask as an integer with the bits that are set to 1 corresponding to
the buttons that are currently pressed down. Bit 0 (the least significant bit) corresponding to
button 1 being pressed, bit 1 corresponding to button 2 and so on.
Return:
An integer mask containing a bit mask of the buttons currently pressed in the specified
canvas.
Arguments:
e
A valid Evas canvas handle.

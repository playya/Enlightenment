void evas_get_color(Evas e, Evas_Object o,
int *r, int *g, int *b, int *a);
This function returns the colour set on an object and writes the colour values into the
pointers passed into the function.
Arguments:
e
A valid Evas canvas handle.
o
A valid object handle in the canvas specified.
r, g, b, a
Pointers to integers where the values for red, green, blue and alpha respectively will be
written.

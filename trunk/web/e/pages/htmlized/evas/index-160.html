Evas_List evas_list_remove_list(Evas_List list, Evas_List remove_list);
This function removes The tail of a list starting at the list pointer
remove_list
from the
list
list
passed in, if it exists, and returns the modified original list.
Return:
A modified list handle.
Arguments:
list
A filled or empty list handle (an empty list being a NULL pointer).
remove_list
The list element pointer to remove from the list as the head of the new list. All elements after
this element in the list are now at the tail of the newly removed list.

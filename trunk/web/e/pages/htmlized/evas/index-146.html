void evas_event_move(Evas e, int x, int y);
This function passes a mouse motion event to the Evas canvas specified, in output pixel
co-ordinates. This should be used if the mouse buttons are pressed or not. As with all the
event functions, callbacks are synchronous and will be called if they are triggered the
moment this function is called, and it will not return until all callbacks triggered have
returned.
Arguments:
e
A valid Evas canvas handle.
x, y
Pixel space co-ordinates where the mouse button has moved to.

Evas_Object evas_add_text(Evas e, char *font, int size, char *text);
This function adds a text string object to the Evas canvas specified. The object by default is
at 0,0 with a default colour of 0,0,0,0. You still need to move the object where you want it
and to show it etc.
Return:
A valid Evas object handle.
Arguments:
e
A valid Evas canvas handle.
font
A font name used to specify the font to use.
size
The font size in Evas canvas units (integer sizes only).
text
A pointer to C-string (0 byte terminated array of chars) array that is used to contain the text
string to use.

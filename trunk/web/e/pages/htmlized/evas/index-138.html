double evas_text_get_inset(Evas e, Evas_Object o);
This function returns the inset from the left side of the text object and where the horizontal
font specified starting co-ordinate is. If the object is not a text object, 0 is returned.
Return:
A value in canvas units that is the inset between the left edge of the font object and the start
of the text.
Arguments:
e
A valid Evas canvas handle.
o
A valid object handle in the canvas specified.

void evas_set_output_size(Evas e, int w, int h);
This routine will tell Evas the size of its output drawable or image in pixels. You must call
this before you call any rendering functions and the width and height (
w
and
h
) must be the
actual size of the drawable as much rendering depends on this being correct.
Arguments:
e
A valid Evas canvas handle.
w
The width of the output drawable or image in pixels. Valid values are between 1 and 8192.
h
The height of the output drawable or image in pixels. Valid values are between 1 and 8192.

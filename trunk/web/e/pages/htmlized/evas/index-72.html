Evas_Object evas_add_gradient_box(Evas e);
This routine adds a gradient box object to an Evas canvas. By default there is no gradient
set and it has an angle of 0 degrees (from 12 noon, clockwise), and at location 0,0 with a
size of 1 x 1 units. You need to set a gradient on this object for it to render anything useful,
and need to move, resize, show etc. the object.
Return:
A valid Evas object handle.
Arguments:
e
A valid Evas canvas handle.

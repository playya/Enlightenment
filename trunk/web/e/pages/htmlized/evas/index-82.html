void evas_set_gradient(Evas e, Evas_Object o, Evas_Gradient grad);
This function sets the gradient to be used by an Evas gradient object. The gradient must be
a valid gradient with 2 or more colours in it and a total spacing of 1 or more across the whole
gradient. The object must be a gradient object, or nothing will happen.
Arguments:
e
A valid Evas canvas handle.
o
The Evas object handle of a gradient object to set the gradient of.
grad
A valid gradient handle to be used as the gradient for the object.

void evas_pointer_pos(Evas e, int *x, int *y);
This function returns the pointer position that the canvas considers the pointer to currently
be at. The output pixel positions are written into the pointers to the co-ordinates provided.
Arguments:
e
A valid Evas canvas handle.
x, y
Pointers to 2 integers that will be filled in with the co-ordinates of the current cursor position
in the specified Evas canvas. If any pointer is NULL it will not be filled in.

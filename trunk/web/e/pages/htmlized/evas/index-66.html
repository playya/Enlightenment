void evas_del_object(Evas e, Evas_Object o);
This function deletes an object from the Evas canvas. This will mean the object dissapears,
and any objects it clips will no longer be clipped by it and will this appear as if they were
normally drawn.h
Arguments:
e
A valid Evas canvas handle.
o
The object to be deleted.

Evas_Object evas_get_object_under_mouse(Evas e);
This function returns a handle to the object that is currently under the mouse pointer position
that is currently held in the Evas canvas. If there is no object under the cursor position,
NULL is returned.
Return:
A valid object handle of an object under the pointer, or NULL if there is none.
Arguments:
e
A valid Evas canvas handle.

Evas_List evas_list_append_relative(Evas_List list, void *data,
void *relative);
This is one of the list convenience functions in Evas. This function appends a pointer to data
to an existing list after the element with the data pointer specified by
relative
. If the element
does not exist in the list the item is appended to the end of the list. Example usage:
Evas_List list = NULL;
void *data, *relative;
list = evas_list_append_relative(list, data, relative);
Return:
A modified list handle.
Arguments:
list
A filled or empty list handle (an empty list being a NULL pointer).
data
A pointer to the data to be appended.
relative
A pointer to the data to be appended immediately after.

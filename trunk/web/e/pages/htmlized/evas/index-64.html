Evas_Object evas_get_clip_object(Evas e, Evas_Object o);
This routine will return the object that clips the specified object. If no object clips this object,
NULL will be returned.
Return:
A valid Evas object handle that is the object that is currently clipping the object specified in
the arguments. If no object is currently clipping the nominated object, NULL is returned
Arguments:
e
A valid Evas canvas handle.
o
The object you wish to query to find its clipping object.

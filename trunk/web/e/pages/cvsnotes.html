E17 CVS Build Notes

After getting E17 from CVS you'll go through the length task of building each of its dependancies. Below you'll find an ordered list that will help you build E. Please note that when we say "build" we're talking about a four stage process of: ./autogen.sh, ./configure (if autogen.sh doesn't run it for you), make, make install. If all goes well you should shoot right through the list. If you have previously installed E17 and its libs your order will be different, this list is for fresh installs only. It is generally recommended that any existing installations of E17 or any of its components be completely removed prior to a build, to avoid the problem we often see where the wrong versions of the libraries get linked.

E17 CVS Build Order:

  1. Install FAM
  2. Build Imlib2
  3. Build Edb
  4. Build Imlib2_loaders
  5. Build EVAS
  6. Build EWD
  7. Build Ebits
  8. Build ECore
  9. Build Etox
  10. Build EWL
  11. Build Etcher
  12. Build Esmall
  13. Build libxml2
  14. Build EFSD
  15. Build PCRE
  16. Build Ferite
  17. Build E.

After the build is complete you should make sure FAM is running, edit your .xinitrc and startx.

Currently the only place you can seek support with E17 build issues is in the #e IRC channel. Don't contact the developers directly unless you are positive you've found a bug! Thanks.

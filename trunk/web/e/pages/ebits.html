Ebits

The Ebits library provides layout functionality for graphical elements like window borders and is based on Evas. It defines the behavior of GUI elements when they are resized, callbacks to call when elements are activated and more. The GUI elements are stored in Edb databases.

Ebits developers include: Raster, Carlos Puchol, and Mark Bainter. Their contact details can be found on the team page.

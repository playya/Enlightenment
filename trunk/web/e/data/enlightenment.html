Enlightenment DR17: The Window Manager


On Nov 24th 2004 Enlightenment DR17's codebase was checked into our CVS repository.

The 17th Development Release (DR17) of the Enlightenment Window Manager brings together more than 3 years of development on its underlying technology: the Enlightenment Foundation Libraries. Imlib2 provides the fastest image proccessing of any general purpose C graphics library avalible. EDB and EET provide unparalleled ease for componant packaging (EET) and database storage (EDB). EVAS provides the most powerful and flexable C canvas library on the planet. Ecore provides robust event management and an extensable framework for general functionality such as X and LinuxFB abstraction, IPC, Sockets, and more. Edje provides the worlds most revolutionary interface abstraction capabilities including advanced visual effects and full animation. Embryo extends Edje's capabilties by adding scripts directly to the interface descriptions. By leveraging all of these componants and more DR17 allows for more functionality, more eye candy, more customization, and more flexability than any other window manager in history. Each release of the Enlightenment Window Manager has set the standard and the DR17 release will push the desktop years beyond what is expected today.

In order to build the DR17 window manager you'll need to install the Enlightenment Foundation Libraries. Feel free to browse the CVS tree but please read the build notes before seeking help.

For those who choose to run Enlightenment DR17, icons can be added by untaring /usr/local/share/enlightenment/data/other/favorite_apps.tar.gz into ~/.e/e/applications/favorite and then copy over the EETs that you want on the iconbar into the ~/.e/e/applications/bar directory.

Miscellaneous Applications

Enlightenment is of course comprised of many libraries. These libraries provide a lot of functionality, and the opportunity to develop some interesting, if not amusing, smaller programs. This page will attempt to list some of those.

To have your E-based application added to this page, let us know on the mailing list.

Entice
Author: Raster
Download: http://www.nl.rasterman.com/projects/entice/index.html
Description: An example image viewer based entirely around the evas library.
 
E_Snake
Author: Martin Sarajervi
Download: http://ms.klmoberg.no/e_snake.tar.gz
Description: An evas-based implementation the so-called "snake" game.

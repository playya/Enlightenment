Ewl

The Enlightenment Widget Library. A simple to use general purpse widget library which is built on all of the the E foundation of subsystem libraries.

This library is still in HEAVY development, but in time will grow to be the final ingredient to rounding off the E development tools.

For just a taste, here is the most basic of examples for EWL:

#include <Ewl.h>

/* Define the main window handle to EWL */
Ewl_Widget *main_win;

int
main(int argc, char **argv)
{
        /* Define a button handle to EWL */
        Ewl_Widget *button;

        /* Initialize EWL */
        ewl_init(argc, argv);

        /* Setup and show the main window */
        main_win = ewl_window_new();
        ewl_window_set_title(main_win,
                             "The Cuddletech Test App: EWL");
        ewl_window_resize(main_win, 297, 68); 
        ewl_widget_show(main_win);

        /* Setup and show a button */
        button = ewl_button_new("Kool Button");
	ewl_object_set_custom_size(EWL_OBJECT(button), 100, 17);
        ewl_container_append_child(EWL_CONTAINER(main_win), button);
        ewl_widget_show(button);

        /* Go into the main EWL Event Loop */
        ewl_main();

        return 1;
}

Functionality is being built in to EWL all the time, and is not currently complete. EWL's primary developers are Christopher 'smugg' Rosendahl and Nathan 'RbdPngn' Ingersoll. Their contact details can be found on the team page.

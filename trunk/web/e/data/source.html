The Source

All of Enlightenments developement occurs directly in CVS. You can pull the code directly from CVS using the following CVS commands: 

cvs -d:pserver:anonymous@@cvs.enlightenment.sourceforge.net:/cvsroot/enlightenment login
cvs -z3 -d:pserver:anonymous@@cvs.enlightenment.sourceforge.net:/cvsroot/enlightenment co modulename

Use the following as module names:

Module nameContent
e17E 17's components
e16E 16's sources
etermEterm's sources
miscMiscellaneous tools
webThe website

Our CVS server is hosted by SourceForge. You can get more info about using CVS in their CVS HOWTO, or browse the Enlightenment CVS Source Trees.

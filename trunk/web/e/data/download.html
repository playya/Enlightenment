Enlightenment v0.16.5, DEB package
Enlightenment support files (required), DEB package

There are a few other required files, but many people likely already have them. If not, either use the "apt-get" method below, or check the Debian package maintainer's page here.

An easier way is to simply add this line:

@:pre@deb http://people.debian.org/~ljlane/downloads potato/@:_pre@

to your /etc/apt/sources.list configuration file and let apt-get do most of the work of getting the needed files.

Eterm is considered Enlightenment's official "terminal emulator", but it certainly works with most, if not all, window managers out there. The website for Eterm is here, Eterm.org.

Here are RPM and source packages of it:

Finally, Enlightenment is probably considered the most visually appealing window manager out there. One way it does this is with "themes", which it includes a couple of with it's regular packages. If you'd like to try different ones out, one of the best sites for finding new ones is e.themes.org.

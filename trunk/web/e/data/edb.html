Edb

This is a database conveneince library based on db 2.7.7 from http://www.sleepycat.com (see src/LICENSE for license details). It is intended to make accessing database information portable, easy, fast and efficient as well as bypass the proplem of libdb continuously changing formats.

Various componants of Enlightemnet use Edb for information storage, including Ebits and the window managers configuration.

Edb's primary developer is Rasterman.

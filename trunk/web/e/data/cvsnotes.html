E17 Build Notes

After getting E17 from CVS you'll go through the length task of building each of its dependancies. Below you'll find an ordered list that will help you build E. Please note that when we say "build" we're talking about a four stage proccess of: autogen.sh, ./configure (if autogen.sh doesn't run it for you), make, make install. If all goes well you should shoot right through the list. If you have previously installed E17 and it's libs your order will be diffrent, this list if for fresh installs only.


    Build Order:
  1. Install FAM
  2. Build Imlib2
  3. Build Edb
  4. Build Imlib2_loaders
  5. Build EVAS
  6. Build EWD
  7. Build Ebits
  8. Build ECore
  9. Build Etox
  10. Build EWL
  11. Build Etcher
  12. Build Esmall
  13. Build libxml2
  14. Build EFSD
  15. Build PCRE
  16. Build Ferite
  17. Build E.

After the build is complete you should make sure FAM is running, edit your .xinitrc and startx.

Currently the only place you can seek support with E17 build issues is in the #e IRC channel. Do not contact developers unless you are positive you've found a bug.

Engage

Engage is a It can launch applications, notify if applications are running, contain the icons of iconified windows and maintain a "system tray" for applications supporting the standard. Another feature is that it can embed other "edje" files such as the clocks pictured below. Engage really is the tidy way to manage all of your applications.

Here is a shot of engage running with each application management feature being used.

The icons in engage are compatible with those that Rasterman is creating for the latest e17 code. This is the first step of the e17 integration that is planned

Engage is built on top of the EFL and fully Edje themeable.

Engage's project manager is Andrew "HandyAndE" Elcock.
